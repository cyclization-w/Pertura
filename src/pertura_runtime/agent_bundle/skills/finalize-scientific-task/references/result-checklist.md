# Result Checklist

- Identity matches the current task and dataset.
- Status is `completed` or `blocked`.
- Analysis unit reports the independent scientific unit actually used.
- Every listed artifact role has a real file at the declared path.
- Every required table header or JSON field is present.
- Findings cite only produced artifact roles and executed metrics.
- Limitations include unresolved design, missing outputs, small replicate count, and exploratory authority when applicable.
- The result is written before the final response object.
