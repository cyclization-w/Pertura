# Target Reliability

Direct target-gene efficacy is informative only when the target is detectable in the relevant state and assay. Keep direct efficacy separate from signature-level efficacy.

Compare guides individually before pooling them by target. Look for direction consistency, effect heterogeneity, leave-one-guide-out sensitivity, and whether one guide drives the pooled result.

Responder and escape classification describes heterogeneity among guide-bearing cells. It does not prove editing efficiency or mechanism by itself. Check whether responder fractions are stable across independent units and contexts.

A signature learned using the tested perturbation labels is not independent confirmation. Mark label exposure and evaluation-split contact as leakage metadata.
