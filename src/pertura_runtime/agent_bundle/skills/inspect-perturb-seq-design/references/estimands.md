# Perturb-seq Estimands

Choose the scientific quantity before choosing a method.

- Target efficacy: whether guide-bearing cells show a direct or signature-level response consistent with perturbing the intended target.
- Replicated expression effect: a condition contrast over independent sample-level count aggregates.
- High-MOI association: a conditional guide-gene or guide-program association accounting for multiple perturbations.
- Composition effect: a change in state proportions across independent samples, not a cell-level contingency alone.
- State effect: altered mapping, occupancy, or response within a frozen state reference.
- Module or global effect: a derived summary of committed gene- or state-level effects.
- Combination effect: a contrast against an explicit additive, linear, or other prespecified baseline.

Keep guide-level and target-pooled estimands distinct. State whether the objective concerns a gene, target, guide, module, state, sample, or perturbation combination.
