# Interpret Perturb-seq Results

Build conclusions from committed results and preserve the distinction between measurement, derivation, prediction, prior knowledge, and hypothesis.

## Procedure

1. Start from the finalized report or committed result projection, not filenames or exploratory prose.
2. Match every statement to the result scope, status, uncertainty, multiplicity handling, dependencies, and limitations.
3. Separate gene-level expression, target reliability, state mapping, composition, module summaries, and virtual predictions.
4. Describe candidate results as exploratory estimates or signals. Reserve stronger wording for statements promoted from current trusted results.
5. Report conflicting guides, replicates, states, or methods as contradictions rather than averaging them away.
6. Generate at least one plausible alternative explanation for a biological interpretation.
7. Label mechanism explanations as hypotheses unless the measured result directly supports the stated relation.
8. Propose the next experiment that would distinguish the leading interpretation from its alternatives.

Read [effect-interpretation.md](references/effect-interpretation.md) for result-specific reasoning. Read [claim-language.md](references/claim-language.md) before drafting conclusions.
